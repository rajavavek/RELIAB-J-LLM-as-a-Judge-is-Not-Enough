# Statistical analysis plan

- Use paired bootstrap resampling of instance groups for pairwise judge comparisons.
- Use McNemar tests for paired binary accuracy comparisons.
- Estimate effects of judge family, domain, stress type, difficulty, and perturbation condition using mixed-effects logistic regression.
- Report calibration with expected calibration error, maximum calibration error, Brier score, and reliability diagrams.
- Use Holm-Bonferroni correction or false-discovery-rate control for multiple comparisons.
- Preserve domain distribution with stratified bootstrap resampling where possible.
